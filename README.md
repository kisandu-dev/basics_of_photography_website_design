## Project Structure

- `index.html` – Home page with hero, module overview cards, and CTA to the Basics.
- `basics.html` – Mini‑course covering Exposure, Focus & Sharpness, and Composition with “Try this now” prompts.
- `contact.html` – Contact form for questions and feedback.
- `style.css` – Single external stylesheet that defines the dark, film‑like UI, layout, and animations.
- `images/` – Folder for hero and module images (place your own JPEG/PNG files here).

## Features

- Semantic HTML5 structure (`header`, `nav`, `main`, `section`, `article`, `footer`).
- Dark, minimalist design with large hero imagery and spacious layout.
- CSS transitions, hover states, and subtle fade‑in animations.
- Accessibility‑first: high contrast colors, focus outlines, alt text placeholders, and keyboard navigable layout.
